损失函数 ​

   损失函数又叫做误差函数，来衡量预测输出值和实际值有多接近。
  • 逻辑回归的损失函数(要尽可能的小))： L(y^,y)=−[ylogy^+(1−y)log(1−y^)]L(\hat{y},y)=-[y\log\hat{y}+(1-y)\log(1-\hat{y})]L(​y​^​​,y)=−[ylog​y​^​​+(1−y)log(1−​y​^​​)]∵y^∈[0,1]\because\hat{y}\in[0,1]∵​y​^​​∈[0,1]∴\therefore∴ 若y等于1，L=−ylog(y)^L=-y\log\hat{(y)}L=−ylog​(y)​^​​，要使L尽可能小，则y^\hat{y}​y​^​​要大，所以y^\hat{y}​y​^​​无限接近于1 同理，y等于0，L=−log(1−y^)L=-\log(1-\hat{y})L=−log(1−​y​^​​),此时y^\hat{y}​y​^​​无限接近于0
